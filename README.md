# Parcial 2 - Arquitectura de Software

Repositorio para la parte práctica del Parcial 2 de la materia Arquitectura de Software, por Miguel Villegas.

## Descripción del Proyecto

Microservicio desarrollado con **Flask** que recibe un número por URL y devuelve una respuesta JSON con:

- El número recibido
- Su factorial
- Una etiqueta indicando si es "par" o "impar" (el número original)


## Configuración del Entorno

### 1. Clonar el repositorio

```zsh
git clone git@github.com:MiguelVN7/Parcial02_MiguelVillegasNicholls.git
cd Parcial02_MiguelVillegasNicholls
```

### 2. Crear y activar el entorno virtual

```zsh
python3 -m venv venv
source venv/bin/activate
```

### 3. Instalar dependencias

```zsh
pip install -r requirements.txt
```
